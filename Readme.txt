Main(функции/процедуры):
gameTime - процедура для таймера игры. Вызывается каждую секунду боя;
gameStart - запускает главный цикл игры;
Main(переменные):
t - таймер боя;
minuteTxt,secondTxt - минуты и секунды в виде текста для удобства
minute,second - минуты и секунды в виде чисел для вычислений
timeText - таймер, который выводится в иконке во время боя.


sprites(функции/процедуры):
addSprite - функция создает и возвращает спрайт. Вызывается при создании персонажа.
addFire - функция создает и спрайт огня mage.


heroes(классы):
Hero - класс персонажа. Принимает параметры:
sprite - спрайт персонажа, созданный с помощью sprites.addSprite;
name - имя персонажа;
speed - скорость персонажа;
damage - урон, наносимый персонажем;
endurance- кол-во выносливости, которое тратит персонаж за одну атаку;
direction - направление персонажа(R/L);
Процедуры класса Hero:
moveR,moveL,stand,hurt,atackExtra,atack,jump,victory,death;
Переменные:
p1,p2 - персонажи игры.


handler(функции/процедуры):
fireHandler - обработчик огня mage;
shootsHandler - обработчик атак персонажей;
jumpHandler - обработчик прыжка;
fightHandler - обработчик боя, который собирает в себе остальные процедуры.


gameInterface(функции/процедуры):
menuInterfaceHandler - обработчик, который вызывает нужную процедуру отрисовки в зависимости от содержания переменной path; 
Процедуры отрисовки:
drawMenu,drawAbout,drawTutorial,drawChoiceChr,drawChoiceMap;


control(функции/процедуры):
control_1,control_2 - управление двумя персонажами.
spriteLife - отвечает за доигрывание состояния спрайта до конца.
